以下是中南地区濒危汉语方言的列表：

The following is a list of endangered Chinese dialects in the Central South region:

瓦乡话
Wa Xiang Dialect

所处位置：湖南西北部
Location: Northwestern Hunan

濒危程度和保护现状：极度濒危，仅偏远村庄使用
Endangerment: Critically endangered, only used in remote villages

发音特征：音调独特，可能与客家或赣语相关
Phonetic Features: Unique tones, possible Hakka/Gan relation

词汇特征：村庄生活词汇，如“田埂”地方表达
Lexical Features: Village-life terms, local “field ridge”

儋州话
Danzhou Dialect

所处位置：海南儋州市
Location: Danzhou City, Hainan

濒危程度和保护现状：子方言濒危，约40万使用者
Endangerment: Endangered sub-dialect, ~400,000 speakers

发音特征：可能与粤语或闽语相关，音系复杂
Phonetic Features: Possibly Cantonese/Minnan related, complex system

词汇特征：历史移民词汇，如“渔船”变体
Lexical Features: Historical migrant terms, variant of “fishing boat”

迈方言
Mai Dialect

所处位置：海南三亚
Location: Sanya, Hainan

濒危程度和保护现状：严重濒危，约1万使用者
Endangerment: Critically endangered, ~10k speakers

发音特征：可能与粤语相关，音调多样
Phonetic Features: Possibly Cantonese-related, diverse tones

词汇特征：粤语客家混合词汇，如“海口”地方表达
Lexical Features: Mixed Cantonese/Hakka vocab, e.g. “Haikou” usage

大鹏方言
Dapeng Dialect

所处位置：广东深圳大鹏新区
Location: Dapeng, Shenzhen, Guangdong

濒危程度和保护现状：濒危，数千使用者
Endangerment: Endangered, a few thousand speakers

发音特征：客家粤语混合，音调复杂
Phonetic Features: Mixed Hakka/Cantonese, complex tones

词汇特征：渔业围屋词汇，如“海蛎”变体
Lexical Features: Fishing/walled-village terms, variant of “oyster”

韶州土话
Shaozhou Vernacular

所处位置：广东北部韶关
Location: Shaoguan, northern Guangdong

濒危程度和保护现状：濒危，未分类，保护少
Endangerment: Endangered, unclassified, minimal protection

发音特征：平话客家粤语混合，音系复杂
Phonetic Features: Mixed Ping/Hakka/Cantonese, complex system

词汇特征：历史融合词汇，如“山歌”地方表达
Lexical Features: Historically fused terms, e.g. “mountain song” variant

Remote Guangxi Dialect

Location: Northern Guangxi
